# Шпаргалка ЕГЭ №5: Анализ алгоритмов для целых чисел

Общий принцип решения

1. **Цикл `for`** : Перебираем исходные числа в разумном диапазоне (обычно от 1 до 1000, если не указано иное).
2. **Тело алгоритма** **: Внутри цикла пошагово реализуем действия из условия.**
3. **Проверка условия** : Если результат соответствует условию задачи, выводим  N  (или) и прерываем цикл (`break`), так как обычно ищут минимальное значение.

---

Тип 1: Системы счисления (Троичная и др.)

*Пример: Замена цифр в троичной записи, удаление ведущих нулей.*

Алгоритм решения:

1. **Перевод в систему счисления** **:**

* Для двоичной: `bin(n)[2:]`.
* Для остальных (например, троичной): используй цикл  `while n > 0` и остатки от деления на 3.

1. **Манипуляции со строкой** **:**

* Используй метод ` .replace()`. **Важно:** чтобы не заменить одну и ту же цифру дважды (например, 0 на 2, а потом эту же 2 обратно на 0), используй временные символы (например, `'0' -> 'temp'`, `'2' -> '0'`, `'temp' -> '2'`).

1. **Удаление ведущих нулей** : Метод `.lstrip('0')`.
2. **Перевод обратно в 10-тичную** : Функция  `int(string_value, base)`, где `base` — основание системы (3, 2 и т.д.).
3. **Математика** : Вычисли модуль разности `abs(n - r)`.

---

Тип 2: Битовые операции и четность (Двоичная система)

*Пример: Дописывание бит четности (сумма цифр % 2).*

Алгоритм решения:

1. **Перевод в двоичную** : `s = bin(n)[2:]`.
2. **Сумма цифр** : В двоичной системе сумма цифр — это количество единиц. Используй `s.count('1')`.
3. **Дописывание разрядов** **:**

* **Вычисли остаток: `res = s.count('1') % 2`.
* **Приклей к строке: `s += str(res)`.
* **Повтори шаг, если в условии сказано "сделать дважды".**

1. **Сравнение** : Переведи результат `int(s, 2)` и проверь условие (например, `R > 43`).

---

Тип 3: Манипуляции с цифрами числа (Десятичная система)

*Пример: Суммы пар цифр и их склейка в порядке убывания.*

Алгоритм решения:

1. **Разбиение на цифры**: Проще всего перевести число в строку `s = str(n)`.
2. **Математические действия** **:**

* Выдели цифры: `d1 = int(s[0])`, `d2 = int(s[1])`, `d3 = int(s[2])`.
* Сложи нужные пары: `sum1 = d1 + d2`, `sum2 = d2 + d3`.

1. **Сортировка и склейка** **:**

* Создай список из полученных сумм: `lst = [sum1, sum2]`.
* Отсортируй его по убыванию: `lst.sort(reverse=True)`.
* Склей в строку: `res = str(lst[0]) + str(lst[1])`.

1. **Проверка** **: Сравни полученную строку с искомым значением (например,  `if res == '1412'`).
